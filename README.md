**Master the end. Learn the beginning.**

A proof of concept of various games that can use the idea of backchaining to create a "fluency" game.

## Platformer

WASD        - Move
Q / ←       - Turn left
E / →       - Turn right  
↑ / ↓       - Look up/down
SPACE       - Jump
Mouse       - Look (when pointer locked)
